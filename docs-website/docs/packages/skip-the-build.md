# skip-the-build

Core package that evaluates your config and returns export conditions.

## Exports

- `defineSkipTheBuildConfig(config)` helper (optional)
- `evaluateConfig(config)` returns `true` when a skip is allowed
- `getExportConditions(config)` returns condition names when a skip is allowed
- Built-in rules and presets

See the Core API docs for details:

- [Config](../core/config.md)
- [Rules](../core/rules.md)
- [Presets](../core/presets.md)
